# A-B-Test-Analysis-for-Online-Shooter-Game
A/B test analysis of 8.6M records for a shooter game. Features: deep EDA &amp; anomaly detection, statistical testing (Welch’s T-test, 95% CI), and user segmentation via K-Means clustering. Result: Validated 4.1% ARPU uplift and optimized monetization strategy. Built using Python (Pandas, Scikit-learn, Statsmodels).
